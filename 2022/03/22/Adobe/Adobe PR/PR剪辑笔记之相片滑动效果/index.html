PR剪辑笔记之相片滑动效果

发表于 2022-03-22 更新于 2022-03-29

滑动效果制作，首先打开pr。
然后选择左上角的文件，选择新建，旧版标题。如果弹出警告点击确定。
![[Pasted image 20220322235949.png]]![[Pasted image 20220323000206.png]]
然后再旧版标题内制作椭圆方框，接着设置椭圆方框的圆角大小，然后选择右侧面板的纹理进行勾选，接着下拉纹理选项，找到想要放入的图片，进行填充
![[source/微信截图_20220323000528.png]]
![[source/微信截图_20220323000724.png]]
接着进行图片的上下复制，选择图片之后按住alt键 ,直到复制出12个图片为止。
![[source/微信截图_20220323000853.png]]
设置完成后，进行平衡居中设置，然后选择右上角的滚动功能，将功能选择为滚动。（我也不知道为什么解释的这么拗口哈哈哈）
![[source/微信截图_20220323001304.png]]
设置完成后就可以关闭旧版标题界面了，文件再左下角生成，然后将生成的文件拖到导轨上。
![[source/微信截图_20220323001408.png]]
将导轨复制出来三份，然后进行下一步的操作![[source/微信截图_20220323001446.png]]
然后按字母R进行时轴缩放设置。调整三个模板。
![[source/微信截图_20220323001557.png]]
接着将第一个模板进行嵌套操作![[source/Pasted image 20220323002500.png]]
嵌套完成后，找到左侧的素材栏，再空白处右键，选择新建项目→颜色遮罩，选择一个自己喜欢的颜色作为背景挡板即可。也可以默认不选。（即当前画布什么颜色就使用什么颜色。）
![[Pasted image 20220323001748.png]]
然后将三个播放轨全部进行嵌套合为一个即可。剩下的就是再播放进度条中选择关键帧进行放大及缩小操作，然后可以制作多个这种效果的照片，中间过渡做一个转场特效即可。
![[Pasted image 20220323002500.png]]
![[source/微信截图_20220323002627.png]]
