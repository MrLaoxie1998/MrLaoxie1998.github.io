当存储好一个场景之后，某些场景做的不够满意，那么可以编辑一个场景。
比如该场景内显示的是青色。如果不需要青色改成黄色。只需要选中场景使用的这些设备。
![[Pasted image 20230427085444.png]]
选中之后，直接在右侧选中CMY RGB选项，然后找到CMY相关选项， 比如现在看到的青色的数值是100%，那就将它调成0，比如需要黄色，那就将黄色改成100%，改成之后。![[Pasted image 20230427085737.png]]
再去点击Functions下面的记录场景按钮(Record Cue)，点击之后，再去选择正在使用场景的按钮，当按下按钮之后，显示器右侧旁边会有三个选项以供选择，分别是取消、代替(替换、颜色的数值会代替里面所有的内容，只留下颜色的信息在内)、并入(把新的数值信息合并到已有的程序数值内去)。选择合并之后，黄色的信息就会储存到该使用的推杆内
![[Pasted image 20230427085914.png]]
假如位置不满意，那么就修改场景内的灯位置，那么就选择该场景下的灯具，然后去选择Pan Tilt按钮![[Pasted image 20230427090342.png]]
接着去调整你要修改的位置，调整完毕之后，下一步选择Record Cue,按完之后，选择所使用的推杆，按一下蓝色按钮(上面修改颜色的操作也是按一下蓝色按钮),修改完毕之后，如果需要快速进行合并，那么就快速按两下蓝色按钮，如果需要其他功能性操作，那么就要选择屏幕右侧的三个选项去做选择。
![[Pasted image 20230427090909.png]]
如果需要把颜色去掉，那么就选择推杆内的一批灯具，然后去选择Attribute Banks内的CMY RGB按钮，接着将所有颜色打成0，然后修改完毕之后，按一下记录场景按钮，然后快速按两下推杆内的蓝色按钮进行合并。

在编辑一个场景中，如果每次都需要合并的功能，而不需要替换功能，如果不想看到屏幕右侧几个按钮，可以通过设置去取消。
1.首先按住SHIFT键，接着按B键打开用户设置，接着按一下NEXT键去选择，那么B按钮有一个选项就是提示选择-并入或替换。可以切换成总并入。在按一下会切换成素材总并入。
![[Pasted image 20230427092802.png]]![[Pasted image 20230427092651.png]]

如何去更新一个场景和一个素材的内容。

比如现在有一个程序或者是素材运行在舞台上，当忘记了或者不知道运行的是哪些设备，或者是不知道舞台上这些设备在素材窗口里面是哪些设备，可以通过选择如果来选择。
1.先按一下Fixture键，然后选择select if选项(按B键)，选择了之后，然后再按程序(推杆)上面的选择按钮(蓝色按钮),按了以后，当前设备所使用的灯具都会在设备窗口屏幕上显示出来。
![[Pasted image 20230427093747.png]]

当选择了程序里面正在运行的设备之后，这个时候可以通过属性选择按钮选择相关的功能。比如改颜色，将黄色改成粉红色，选择CMY RGB，然后将黄色Yellow变成0，然后调整Magenta，调整到100%,变成粉红色，这个时候选择记录场景按钮(Record Cue)之后，在屏幕右侧的B键上，有一个更新选项，按了之后，屏幕上所有的设备包括素材都变成了灰色，只有使用中推杆上的素材和程序是高亮显示的。这个时候需要确定更新哪个内容：
1.如果是更新正在使用中的推杆的话，那么只需要按一下ENTER确定键，这个所更改的(粉红色)，会马上更新到所使用的推杆内。
2.如果是更新素材，可以只选中素材，然后取消场景，然后再按确定，那么这个粉红色会马上更新到素材内。
3.所有操作完成后要记得清空编辑器(按CLEAR)
![[Pasted image 20230427094138.png]]
![[Pasted image 20230427094637.png]]

如果保存好一个程序之后，想查看这个程序都保存了什么内容

运行一个程序(推杆)，如果控制台没有连接到舞台，想看程序保存了什么数值。 这个时候，需要按程序的VIEW（查看）键，按下之后选择这个程序(推杆)的选择按钮，选择了之后，在外接显示器这里，会弹出一个（回放程序查看）这样一个窗口。 里面有如下信息：
1.Delay In(渐入)
2.Fade In(淡入)
3.Fade Out(淡出)
4.Fixture Overlap(设备交叠)
5.Curve(曲线)
6.View Cue(查看提示)
![[Pasted image 20230427110135.png]]

然后点击View Cue下的View按钮，就可以弹出Cue查看窗口。可以通过右侧的菜单去查看想要看的内容，可以点击数值显示灯具的所有数值信息。
