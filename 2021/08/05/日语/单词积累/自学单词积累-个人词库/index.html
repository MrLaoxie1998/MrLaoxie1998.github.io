自学单词积累-个人词库

发表于 2021-08-05 更新于 2021-08-18 分类于 日语 ， 日语单词

这是我自己积累的单词，虽然其他单词也有记录，但是这是每个单词下面都有一个例句，为了加深自己对单词的记忆能力

单词、例句 词性 翻译 声调
食(しょく) 名词 饮食 0
日々の食はあなたの健康に強い影響を与えています。 日常的饮食严重影响着你的健康。
食卓(しょくたく) 名词 餐桌 0
夜、家族で食卓を囲み今日一日あったことを話し合います。 晚上，全家人围坐在餐桌旁，聊着各自当天的见闻。
コロッケ 名词 炸肉饼 1
父はコロッケにソースをたっぷりかけて食べるのが好きです。 爸爸喜欢在炸肉饼上浇很多酱汁来吃。
あー 感叹词 啊 1
一日ずっと歩き(あるき)っぱなしで足が痛い。あー、早くこの靴脱ぎたい。 整整走了一天，脚真疼。啊，真想赶紧把这鞋脱了。
お兄ちゃん(おにいちゃん) 名词 哥哥 2
年をとっても、兄を呼ぶ時は、やはり「おにいちゃん」と言ってしまいます。 虽然自己岁数大了，但称呼兄长时还(和从前一样)叫“お兄ちゃん”
部活(ぶかつ) 名词 课外社团活动 0
大学受験が近いから、もう部活はやめたほうがいいでしょうか。 临近高考了，是不是暂时就别参加课余活动了。
ダイこング 名词 食堂，餐室 1
このダイこングは広いのでパーティーをするのに向いています。 这餐厅挺大的，适合聚餐、聚会。
おっ 感叹词 哦 1
おっ、今日のヘアスタイル悪ないね。 哦，今天的发型不错啊。
一家(いっか) 名词 一家，一个门户 1
毎年の元旦には一家そろって神社に初詣にいきます。 每年元旦全家都会一起参拜神社，辞旧迎新。
チキン 名词 鸡肉，鸡 1
息子はチキンが大好きで、毎日食べても飽きないようです。 我儿子特别喜欢吃鸡肉，天天吃都不会腻似的。
ラッキョウ 名词 蕌头 0
ラッキョウは甘酸っぱくて、カレーの辛さによく合いますね。 （腌制的）蕌头酸甜可口，和咖喱的辛辣还挺相配的。
福神漬け(ふくじんづけ) 名词 福神腌菜(什锦八宝酱菜) 0
余った野菜はお醤油と砂糖に漬けて福神漬けにしましょう。 剩余的蔬菜就用酱油和砂糖来腌制什锦八宝酱菜吧。
差し出す(さしだす) 动词1类、他动词1类 拿出、伸出 3
彼女は窓から手を差し出して、落ちてくる雪を受け止めようとしました。 她把手伸出窗外，想用手去接从天而降的雪花。
おかわり 名词、サ　　变他 再来一盘(一份) 2
健康のためにご飯のおかわりは2杯までにすることにしました。 我决定，为了身体健康，就餐时米饭最多吃两碗。
めんた 名词 你(比较随意的说法) 1
という言葉は親しい間柄で個人的場面(ばめん)で呼ぶ時に使うので、使う(つかう)場面を考えなくてはいけない言葉(ことば)です。 “あなた”这个词是比较亲近的人之间私下里称呼用的，使用时必须考虑场合和对象。
ドリブル 名词、サ变他 运球 0
彼はドリブルをしながら誰よりも早くゴール下に行くことができるんです。 他能做到运球时快速移动，先于别人到达篮下。
腹減った(はらへった) ? 肚子(饿了) ?
1時間前に食べたのに、もう[腹減った]ですか。 1小时前刚吃过，怎么又喊”饿了”?
マリー(法语Marie) 固有名词 (人名)玛丽 1
まさか、貴方の名前はマリちゃん？ 难道，你叫玛丽酱？
チャリヤー(泰语carya) 固有名词 (人名)恰丽娅 1
私は名前はチャリヤーです。 我的名字叫恰丽娅
ツアー 名词 旅行/团体旅游 1
回遊旅行。団体旅行。遠足。小旅行。 旅游。郊游。团体旅行。短途旅行。
東大寺(とうだいじ) 固有名词 (地名)东大寺 1
東大寺の建て方は中国の寺の建て方と似ています。 东大寺的建筑方式和中国的很相似。
修学旅行(しゅうがくりょこう) 名词 修学旅行(这是属于日本中小学教育活动的一环) 5
今日は修学旅行に行きましょう。 今天我们一起去修学旅行吧！
座席(ざせき) 名词 座位 0
荷物を座席の下におく。 把行李放在座位底下。
貴重品(きちょうひん) 名词 贵重物品 0
貴重品輸送。 贵重品运送。
興福寺(こうふくじ) 固有名词 (地名)兴福寺 1
ここでは興福寺です。 这里是兴福寺。
カメラ(camera) 名词 照相机 1
カメラにおさめる。 照下来。拍下来。
ガム(gum) 名词 口香糖 1
これはガムですよ。 这个就是口香糖哦。
嚙む(かむ) 他动词1类 嚼，咀嚼，咬 1
ごはんをよく嚙む。 饭要细嚼。
拝観(はいかん) 名词/他动词3类 参观、参拜 0
たからものなどを見ること。 参观。观看。
撮影(さつえい) 名词、他动词3类 摄影/照相 0
記念撮影をする。 拍（照）纪念照。
鑑真(がんじん) 固有名词 (人名)鉴真 1
奈良時代に渡来した唐の僧。日本の律宗の開祖。 奈良时代的舶来僧。日本律宗之祖。
以来(いらい) 名词 以来/以后 1
解放以来の変化が一目でわかる。 新中国成立以来的变化一目了然。
遭難(そうなん) 名词/自动词3类 遇难/遇险 0
火山の爆発で5名遭難した。 因为火山爆发，有5人遇难。
失明(しつめい) 名词 失明 0
視力を失うこと。 失明。
森山(もりやま) 固有名词 森山 1
私の名前は森山です。 我叫森山
